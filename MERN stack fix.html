# backend/.env (use this file)

PORT=5000
NODE_ENV=development

# Database

# Inngest (if used)
INNGEST_EVENT_KEY=your_inngest_event_key
INNGEST_SIGNING_KEY=your_inngest_signing_key

# Stream (required for activity feed features)
STREAM_API_KEY=your_stream_api_key
STREAM_API_SECRET=your_stream_api_secret

# Clerk (auth) (optional: only if your app uses Clerk)
CLERK_PUBLISHABLE_KEY=your_clerk_publishable_key
CLERK_SECRET_KEY=your_clerk_secret_key

# Frontend origin (used for CORS / redirects)
CLIENT_URL=http://localhost:5173

Notes on values

  • Replace your_mongodb_connection_url with your actual MongoDB connection string (or leave blank if you dont use Mongo).

  • Replace every your_* placeholder with the real secret keys from the provider dashboards.

  • PORT=5000 is common for backends (your earlier example had 3000 both are fine, but choose one and make sure frontend expects it).

  • NODE_ENV=development is optional but useful for conditional behavior in code (dev vs prod).
